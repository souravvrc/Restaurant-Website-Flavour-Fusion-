# 🍽️ Flavour Fusion – Restaurant Website

A modern, responsive restaurant website built using HTML, CSS, JavaScript, Node.js, Express.js, and MongoDB. The project provides an engaging user experience with an interactive menu, online table reservation system, contact page, and responsive design for seamless browsing across devices.

---

## 📌 Project Overview

Flavour Fusion is a full-stack restaurant website designed to simulate a real-world restaurant's online presence. The website allows customers to explore the menu, learn about the restaurant, make reservations, and get in touch through an intuitive and visually appealing interface.

The project demonstrates front-end web development, backend integration, database connectivity, and responsive UI design.

---

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/souravvrc/restaurant-website-flavour-fusion.git
cd Restaurant-Website-Flavour-Fusion
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Start the Server

```bash
node server.js
```

or (if using Nodemon)

```bash
nodemon server.js
```

### 4. Open the Website

Visit:

```
http://localhost:3000
```
